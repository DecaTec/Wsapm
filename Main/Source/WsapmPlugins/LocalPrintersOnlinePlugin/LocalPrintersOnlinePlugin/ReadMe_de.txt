Lokale Drucker Online Plugin
v1.1.0

Dieses Plugin prüft, ob ein lokaler Drucker angeschaltet ist und unterdrückt ggf. den Standby-Modus.

Plugin-Einstellungen:
- Zeigt eine Liste der verfügbaren Drucker. Wenn mindestens einer der ausgewählten Drucker online ist, wird der Standby-Modus unterdrückt.
