Twonky Streaming Status Plugin
v1.0.0

Dieses Plugin ermittelt den Streaming-Status eines Twonky Streaming-Servers.
Wenn gerade ein Medien-Stream aktiv ist, wird das Welchseln des Computers in den Standby-Modus unterdrückt.

Plugin-Einstellungen:
- URL des Twonky-Interfaces: URL des Twonky Web-Interfaces. Wenn Twonky gestartet wurde, wird das Web-Interface über das Twonky-Symbol in der Schnellstart-Leiste aufgerufen (standardmäßig 'http://127.0.0.1:9000/').
