Twonky Streaming Status plugin
v1.0.0

This plugin enables Windows Server Advanced Power Management to check the streaming status of a Twonky Server.
If Twonky is currently streaming media, the computer's standby mode will be suppressed.

Plugin settings:
- URL of Twonky web interface: This is the URL of the Twonky web interface. Twonky adds a status bar symbol where the web interface can be called. This is usually 'http://127.0.0.1:9000/'.
